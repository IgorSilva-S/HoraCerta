Como criar o seu tema


Introdução

O aplicativo Hora Certa permite que o usuário faça um tema do jeito que ele quiser, com sua própria fonte, cor e aparição da linha. Porém, para ser utilizado, é necessário saber utilizar um pouco o Google Fonts. Nesta janela, iremos ensinar como você pode selecionar os arquivos do site e utilizá-los no seu Widget da forma que quiser, ou, caso prefira, usar as fontes de outros temas e customizar do seu jeito.

Requisitos

  1. Sempre manter o widget conectado à internet
  2. Paciência
Observação: O tema customizável pode ser utilizado com o horário local ativado, mas não pode ser utilizado sem internet.

Encontre a fonte que irá utilizar

Acesse o Google Fonts e procure pela fonte que deseja utilizar no widget. Ao escolher, aperte no botão Get Font. Após isso, você será direcionado para uma página com o nome da fonte e dois botões. Ao entrar nessa página, aperte o botão Get Embed Code.

Embed Code

Pode parecer assustador, mas é super simples passar por essa etapa. Ao apertar o botão, você será direcionado para a página de códigos do site. Na guia Web, selecione a opção com o nome @import. Selecionando, copie o link que está dentro das aspas dentro dos parênteses na seguinte linha:
CSS
@import url('https://fonts.googleapis.com/css2?family=Sigmar&display=swap');

/*Esse link é de exemplo para a fonte Sigmar.*/
Você vai copiar o link dentro das aspas e colar na caixa de texto que pede a URL da fonte. Após isso, você vai voltar ao site e verá outra parte com texto escrita (usando o exemplo da fonte Sigmar): Sigmar: CSS class, será algo semelhante ao escrito na caixa abaixo:
CSS
.sigmar-regular {
font-family: "Sigmar", serif;
font-weight: 400;
font-style: normal;
}
Você vai pegar o nome da fonte que está após os dois pontos de font-family, no caso do exemplo " "Sigmar", serif " (não copie o ponto e vírgula) e colar na caixa de texto que pede o nome da fonte.

Alterações

Após esse processo, faça as alterações que quiser e salve o tema. Caso o processo tenha sido feito corretamente, o widget apresentará a fonte escolhida do Google Fonts!

Fontes locais

Gostou das fontes dos temas que já existem, mas quer alterar tamanho, presença de linha e outros? Você pode usar essas fontes sem precisar informar a URL do Google Fonts aqui está uma lista com o nome das fontes utilizadas nos temas:
  • "Segoe UI Variable", do tema Windows 11
  • "Segoe UI", do tema Windows 10
  • "One UI Sans", do tema One UI
  • "Jersey 10", do tema Pixels
  • "DS-Digital", do tema Wii Menu (Parte de cima)
  • "Segoe UI", do tema Wii Menu (Parte de baixo)
  • "Andes", do tema Sonic 3 & Knuckles
  • "Lovelo", do tema Linhas
  • "Great Vibes", do tema Royal
  • "Times New Roman", do Tema Customizável (antes de customizar)
Para usar qualquer uma dessas fontes, basta digitar o nome entre aspas (com as aspas) na caixa de texto que pede o nome da fonte e você poderá usar elas como quiser no seu tema.
